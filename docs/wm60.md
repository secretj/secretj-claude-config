# wm60 컨텍스트

## 역할
구버전 메일 및 로그인 SSR (CI3)

## 디렉토리 구조
```
wm60/
├── app/                    # CI3 구조 (controllers, models, libraries, helpers, hooks, config, core, views, language, cache, logs, third_party)
├── api/                    # 독립 PHP 스크립트 (SSR)
│   ├── api_auth.php        # 인증
│   ├── api_pop3_new.php    # POP3
│   ├── api_user_set.php    # 사용자 설정
│   ├── api_mail_get.php    # 메일 조회
│   ├── api_delete_gsuite_user.php  # G Suite 사용자 삭제
│   ├── dbs_sync.php        # DB 동기화
│   ├── domainDir.php       # 도메인 디렉토리
│   ├── eml_getdata.php     # EML 데이터
│   ├── notify.php          # 알림
│   ├── sso.html            # SSO
│   ├── v1/                 # v1 API
│   └── v2                  # v2 API (심볼릭)
├── module/
│   └── mail/               # 메일 모듈
├── do/
│   └── mailPush.php        # 메일 푸시
├── asset/                  # 정적 자원
├── editor/                 # 에디터
├── lang/                   # 언어팩
├── views/                  # SSR 뷰 템플릿
├── .nginx_rules.conf       # nginx 규칙
├── index.php               # 엔트리포인트
└── version                 # 버전 파일
```

## 주요 작업 이력

### 핑거프린트 쿠키 개선 (WM-30174)
- 아이디에 @ 2개 이상 → 마지막 @로 도메인/아이디 분리
- 쿠키 보관 기간 10년 설정
- /api/v2/mist/source 요청 시 FP 발급 대상 여부 리턴 → 불필요한 요청 감소

### 로그인/보안
- DRYRUN flag 로직 수정 (LS-317)
- 의심 IP 차단 케이스 추가 (LS-481)
- 2단계 인증 OFF인데 OTP 화면 노출 이슈 (LA-3827)
- return_url 허용 도메인 아닌 경우 redirect 제한 (WM-29450, Holding)

### 내부메일함
- 구버전에서 내메일함 하위 '내부메일함' 생성 제한 (WM-32591)
- 내부편지함 이름 변경 (WM-32758, 진행중) — batch 스크립트 작업

### 구 소스 분리 (WM-32209, Holding)
- api - 통합디비 → 웹메일 동기화 (WM-30695)
  - wm70-lib: dbs_sync 큐동기화 변경

- api - pop3/imap/smtp 인증 (WM-30691)

### 관리자 기능
